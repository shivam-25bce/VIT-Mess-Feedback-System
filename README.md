# VIT-Mess-Feedback-System
# Problem Statement
In many institutions, collecting and analyzing student feedback regarding mess food is often done manually, which is time-consuming, unorganized, and prone to errors. There is a lack of a structured and efficient system to gather feedback and derive meaningful insights such as average ratings and poorly performing days. This project aims to solve this problem by providing a simple, automated, and user-friendly console-based application that allows students to submit feedback easily and helps in systematically storing and analyzing the data for better decision-making and improvement of mess food quality.

The Mess Feedback System is a simple Python-based console application designed to collect and analyze feedback from students regarding mess food quality. The system includes a login feature where users authenticate using predefined usernames and passwords. After login, users can submit feedback by rating taste, hygiene, and quantity on a scale of 1 to 5. Each feedback entry is stored as a dictionary in a list.
The application provides multiple options through a menu, including submitting feedback, viewing personal feedback, calculating overall average ratings, and identifying the worst food day based on user inputs. It ensures basic input handling and organizes data efficiently in memory.
This project demonstrates core Python concepts such as functions, loops, conditionals, lists, dictionaries, and user input handling. It can be further enhanced by adding file storage, GUI interface, and database integration for scalability.
